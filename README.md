# TaskFlow — Full-Stack Task Management App

A full-stack task management application built with **Vue.js** on the frontend and **Node.js + Express** on the backend.  
TaskFlow allows users to create, view, and manage tasks with a clean UI and RESTful API architecture.

---

## Features
- Create tasks
- View task list
- Backend REST API
- Frontend–backend integration using Axios
- Modular project structure
- Local development setup

---

## Tech Stack
### Frontend
- Vue 3
- Vite
- Axios

### Backend
- Node.js
- Express
- CORS
- Nodemon
